Recriando o jogo da cobrinha com JavaScript

Projeto desenvolvido durante as aulas da Digital Innovation One sobre a recriação do jogo da cobrinha 

O que acrescentei: 

Espaçamento entre o h1 e canvas
Borda ao redor do canvas
Pontuação para cada vez que a cobrinha "pegar" a comida. No final do jogo, a pontuação é mostrada.
Assim que a pontuação é mostrada, o usuário clica em ok e poderá escolher se quer ou não continuar jogando selecionando ok novamente, para continuar, ou cancelar, para não jogar.
